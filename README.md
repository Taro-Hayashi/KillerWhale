## ご注意
- 左右分割キーボードとして使う場合、TRRSケーブルの抜き差しはUSBケーブルを繋いでいない時に行なってください。
- 頻繁にユニットを付け替えるとピンソケットがゆるんで接触が悪くなることがあります。
